Exercise: View the Search Developer's Guide

In this walkthrough exercise, you view the MarkLogic Search Developer's Guide.

 

  1. In a browser, navigate to docs.marklogic.com/guide/search-dev.pdf, to open the Search Developer's Guide.
  2. Scroll down through the table of contents, scanning it as you go.
  3. Skim pages 15-19, especially noting the diagram on page 16.
  4. Continue perusing the guide, as desired.

View the search documentation for the Node.js API and the Java API (covered in the respective developer courses):

  1. Navigate to docs.marklogic.com, to go to the MarkLogic documentation landing page.
  2. In the Developer's Guides section, click the Node.js Application Developer's Guide link.
  3. See that there is a Querying Documents and Metadata chapter.
  4. Click the browser's Back button, to return to the MarkLogic documentation landing page.
  5. Click the Java Application Developer's Guide link.
  6. See that there is a Searching chapter.
  7. Continue perusing, as desired.
