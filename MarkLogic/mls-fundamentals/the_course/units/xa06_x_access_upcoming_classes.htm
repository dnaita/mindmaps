Exercise: Access Upcoming Classes

In this walkthrough exercise, you view upcoming MarkLogic University classes.

 

  1. In a browser, navigate to mlu.marklogic.com.
  2. Notice the Developer Track, the Administrator Track, and the On-Demand Classes:

  3. Select a track of interest to you.
  4. Scroll down through the list of classes, noting ones of interest to you.
  5. Click the Learn More button for a class that interests you (e.g. Developing MarkLogic Applications I - XQuery).
  6. Read the course description, to see if it would be an appropriate course for you.
  7. If desired, click the View Schedule button, to see the date(s) and time zone for when the class is next being offered.
