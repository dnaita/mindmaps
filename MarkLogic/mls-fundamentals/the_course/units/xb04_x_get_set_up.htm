Exercise: Get Set Up

In this walkthrough exercise, you open the course virtual machine and locate the course files.


  1. In VirtualBox, with the centos_64_fundamentals virtual machine selected, click the Start icon:

  2. Wait for the virtual machine to boot.
  3. Click cent:


  4. For password, enter centos.
  5. Click the Log In button.
  6. See that you are now inside the virtual machine:

  7. On the desktop of the virtual machine, notice the mls-fundamentals folder.
  8. Realize these are the course files.
  9. Double-click the folder, to open it:


  10. Double-click the folder named the_course.
  11. Double-click the file named index.htm, to open it in a browser.
  12. Realize that you are viewing the table of contents for the course:


(Optional) If desired, download the course files outside of the virtual machine, onto your own computer:

  1. In a browser window on your own computer (outside the virtual machine), enter https://mlu.marklogic.com/downloads/ml8-fundamentals-sp.zip to download a .zip file of the course files.
  2. Wait until the ml8-fundamentals-sp.zip file has finished downloading.
  3. Unzip the file.
  4. See that you now have access to the class files (i.e. mls-fundamentals folder) outside of the virtual machine.
