Exercise: Check for Hardware Requirements

In this walkthrough exercise, you check that the computer that you are thinking to use for hosting the course virtual machine (for doing the course hands-on exercises) meets the hardware requirements of 64-bit, 4 GB or more of RAM, and 20 GB or more of free disk space.

Windows | Macintosh | Linux

 

If it is a Windows computer (steps may vary depending on your Windows version):

  1. In Windows Explorer, right-click Computer (not C:), to then select Properties.
  2. See that the System window appears:


  3. Check that the System type is 64-bit:


  4. Check that the installed memory is at least 4 GB:


  5. In Windows Explorer, right-click C:, to then select Properties.
  6. See that the Local Disk (C:) Properties window appears:


  7. Check that the amount of free space is at least 20 GB:

 

If it is a Macintosh computer:

  1. Go to the Apple menu (upper-left corner) and select About This Mac.
  2. Notice the Processor value (e.g. 1.3 GHz Intel Core i5):

  3. Check the Processor value against this list, ensuring that it is 64-bit:

  4. Also in the About This Mac window, check that the Memory value is at least 4 GB:

  5. Select the Macintosh HD.
  6. In the File menu, select Get Info.
  7. Check the Available value to see that the amount of space available is at least 20 GB:

 

If it is a Linux computer:

  1. To check that the operating system is 64-bit, enter getconf LONG_BIT in a terminal window:


  2. See that it says 64 (not 32).
  3. To view the amount of memory, enter free -m:


  4. See that the value in the total column is at least 4096 (i.e. 4 GB):


  5. To check the amount of free space available, enter df -h:


  6. In the Avail column, see that the first value is at least 20G (i.e. 20 GB):
