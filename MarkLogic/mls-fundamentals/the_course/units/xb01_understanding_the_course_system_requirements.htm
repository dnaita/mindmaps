Understanding the Course System Requirements

For hosting the course virtual machine, for doing the hands-on exercises in this self-paced course, you will need a computer with the following system requirements.

Hardware

The student computer needs to be 64-bit, have 4 GB or more of RAM, and have 20 GB or more of free disk space.

Software

The student computer needs a 64-bit operating system, needs the latest version of VirtualBox installed and working, needs Acrobat Reader (or equivalent, for viewing PDF files), and needs a way to unzip files.

Internet

Internet access is required.

 

 

Note: If you are attending this course via the live online or in-person MarkLogic training classes provided by MarkLogic University, you can skip this appendix. You simply need an Internet connection and browser (most browsers and operating systems are supported) to use an online virtual machine provided to you for use during class.
