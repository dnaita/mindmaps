Exercise: Post a Question

In this walkthrough exercise, you see how you could post a question on Stack Overflow. Then you see how you could join the MarkLogic developer mailing list.

 

  1. In a browser, navigate to stackoverflow.com.
  2. In the search field, enter MarkLogic.
  3. Notice that you are viewing questions and answers for MarkLogic.
  4. In the upper-right, click the Ask Question button.
  5. See that you are prompted to log in or sign up:


  6. If you already have a log in, go ahead and log in. If not, select the Sign up tab, to register.
  7. If desired, post a question.
  8. In a browser, navigate to developer.marklogic.com.
  9. Select the Q&A link:


  10. See that the Ask a Question button links to Stack Overflow.
  11. Click the browser's Back button, to return to the MarkLogic Questions and Answers page.
  12. Click the developer mailing list link.
  13. See that you are now on the Mailing Lists page.
  14. Click the markmail.org link, to be able to see and search previous posts to the general development mailing list.
  15. Peruse MarkMail as desired.
  16. Click the browser's Back button, to return to the Mailing Lists page.
  17. Click the general development link.
  18. If desired, fill out the form to register for the general development mailing list.
