Creating a Database

A forest can be created through the Admin Interface by using the Create tab in the Databases section.

Within the Create tab, the only required field is database name, to give the database a name.

Attaching a Forest to a Database

Once a forest and a database have been created, the forest can be attached to the database.

To view, or change, which forests are attached to database, the Databases category on the left can be selected, to then select the name of the database, to then select Forests under it.

Viewing the Status of a Database

The status of a database can be viewed by selecting the database name on the left and then the Status tab at the top.
