Creating the Course Virtual Machine

The virtual machine for doing the hands-on exercises in this course is downloadable from MarkLogic University.

For people curious to know how the virtual machine was created, the steps that were done (using VirtualBox 4.3.28 on Windows in November 2015) are provided in this appendix. (Realize that if you go to reproduce these steps at a later date, some of the open source software versions and URLs may have moved on.)

At a high level, the steps that were done to create the virtual machine for this course were:

  1. Created a CentOS virtual machine.
  2. Configured CentOS.
  3. Installed helper applications.
  4. Installed JDK.
  5. Installed Maven.
  6. Installed Tomcat.
  7. Installed Eclipse.
  8. Installed Node.js.
  9. Installed npm.
  10. Installed Git.
  11. Installed Bower.
  12. Downloaded the class files.
  13. Downloaded the MarkLogic installer.

Note: It is not necessary to build the virtual machine yourself. You can download it from MarkLogic University.
