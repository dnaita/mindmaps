Understanding Search

With search included in the core of MarkLogic, MarkLogic is able to offer powerful search abilities in real-time.

MarkLogic's search capabilities include text search, geospatial search, semantics search and alerting (reverse search).

Search features include facets, type-ahead suggestions, highlighting, snippets, relevance ranking, document structure-based search, proximity boosting, search term weighting, wildcards, stemming, case sensitivity, punctuation sensitivity, and diacritic sensitivity.

In addition to English, MarkLogic offers tokenization, stemming, and collation for Arabic, Chinese (Simplified), Chinese (Traditional), Dutch, French, German, Italian, Japanese, Korean, Norwegian, Persian (Farsi), Portuguese, Russian, and Spanish.

MarkLogic's text search capabilities can also be used with its geospatial search (and/or semantics search) capabilities to create discovery applications, as well as to visualize data. These applications could even include alerting, for notification of any new items.
