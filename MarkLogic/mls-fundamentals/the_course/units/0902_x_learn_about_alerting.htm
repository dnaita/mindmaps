Exercise: Learn About Alerting

In this walkthrough exercise, you learn about alerting.

 

  1. In a browser, navigate to docs.marklogic.com/guide/search-dev/alerts.
  2. Skim through the first few paragraphs, learning that alerting is done through a reverse query, where alerting happens when a new document is inserted in the database that matches a pre-defined query.
  3. If desired, continue reading the guide.
