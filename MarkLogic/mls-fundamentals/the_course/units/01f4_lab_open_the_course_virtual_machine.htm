Lab: Open the Course Virtual Machine

In this lab exercise, you open the course virtual machine and view the course files.

Note: If you haven't already, do the steps in Appendix B, to set up the course virtual machine on your computer.

 

  1. On your computer, in VirtualBox, with the centos_64_fundamentals virtual machine selected (that you previously downloaded from MarkLogic University and then imported), click the Start icon:

  2. Wait for the virtual machine to boot.
  3. Click cent:


  4. For password, enter centos.
  5. Click the Log In button.
  6. See that you are now viewing the desktop of the virtual machine:


View screenshots of the applications we will be working with in this course:

  1. Double-click the mls-fundamentals folder on the virtual machine's desktop.
  2. Notice the unit folders:


  3. Double-click the unit01 folder, to view its contents:


  4. Double-click the star-wars_screenshot.pdf file, to open it:


  5. Scan the screenshot for this application, which uses MarkLogic as the database, Node.js as the middle tier, and Angular for the front-end, created for people to be able to view and search Star Wars characters.
  6. In the menu bar, notice Home, Characters, and Search.
  7. Realize that the data as well as the images for the characters are being retrieved from the database.
  8. Double-click top_songs_screenshot.pdf, to view it:

  9. In this MarkLogic single-tier application, built with XQuery, notice the search interfaces, including the faceted navigation, search field, and advanced search.
  10. Realize that all of the data is being retrieved from a MarkLogic database, including for the facet navigation, search results, and more links.
  11. Close the files.
