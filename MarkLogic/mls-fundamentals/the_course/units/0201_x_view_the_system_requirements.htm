Exercise: View the System Requirements

In this walkthrough exercise, you view the system requirements for MarkLogic Server 8.

 

  1. In a browser, navigate to developer.marklogic.com/products/marklogic-server/requirements-8.0.
  2. Notice the supported platforms, along with the conditions for each.
  3. Scroll down to view the memory and disk space requirements.
