Exercise: Use VirtualBox

In this walkthrough exercise, you set up VirtualBox on your computer, and then then import the course image.


  1. On your computer, in a browser window, go to http://www.virtualbox.org/wiki/Downloads.
  2. Select the appropriate VirtualBox download link, for your operating system:


  3. Follow the prompts to download and install VirtualBox onto your computer.

    Note: If you need help installing VirtualBox for your operating system, see the Installation details chapter in the VirtualBox User Manual at www.virtualbox.org/manual/ch02.html.
  4. Launch VirtualBox.

Import the course virtual machine into VirtualBox on your computer:

  1. In VirtualBox, select File - Import Appliance....
  2. Click the folder icon, to then browse to the location of the centos_64_fundamentals.ova file that you downloaded in the previous exercise.
  3. Select the centos_64_fundamentals.ova file and click the Open button.
  4. Click the Next button.
  5. Select the checkbox for Reinitialize the MAC address of all network cards.
  6. Click the Import button.
  7. See that the virtual machine is now available:
