Understanding the System Requirements

MarkLogic can be installed on the following operating systems:

  • Microsoft Windows Server 2012 (x64) and 2008 (x64)
  • Windows 8 (x64) and 7 (x64)*
  • Red Hat Enterprise Linux 7 (x64) and 6 (x64)
  • CentOS 6 (x64)
  • SUSE Linux Enterprise Server 11 (x64)
  • Sun Solaris 10 (x64)
  • Amazon Linux 2013.03 (x64)
  • Mac OS X 10.9 or 10.8*

*Mac OS, Windows 8, and Windows 7 are for development only.

For RAM, MarkLogic requires a minimum of 512 MB of system memory, with 2 GB or more recommended.

For disk space, MarkLogic requires 1.5 times the size of the loaded source content, to allow for merging.
