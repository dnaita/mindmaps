Creating a Forest

A forest can be created through the Admin Interface by using the Create tab in the Forests section.

The only mandatory field for creating a forest is forest name, to give the forest a name.

The data directory field is for specifying a location for the forest on the host computer. If this field is left blank, the forest will be saved in the default directory. The default data directory for Linux is /var/opt/MarkLogic/Forests. The default data directory for Windows is C:\Program Files\MarkLogic\Data\Forests. The default data directory for the Macintosh is ~/Library/Application\ Support/MarkLogic/Data.

Detail on creating and working with forests is available in the Forests section of the Administrator's Guide, available at docs.marklogic.com/guide/admin/forests.
