Installing MarkLogic

MarkLogic can be installed on CentOS by double-clicking the installer file (e.g. MarkLogic-8.0-3-2.x86_64.rpm) and then following the installation prompts.


Configuring Which User MarkLogic Will Run As in CentOS

By default, on UNIX-based operating systems, MarkLogic runs as a Unix user named daemon.

In CentOS, to change which user MarkLogic will run as (e.g. to provide MarkLogic with the directory and file permissions of a particular user), the /etc/sysconfig/MarkLogic file can be edited.

In non-Unix operating systems (e.g. Windows), it is not necessary to specify a user to run MarkLogic.


Other Platforms

The Installation Guide for All Platforms (docs.marklogic.com/guide/installation.pdf) contains the steps for installing MarkLogic on each supported operating system.
