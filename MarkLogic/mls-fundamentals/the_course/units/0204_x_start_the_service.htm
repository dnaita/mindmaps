Exercise: Start the Service

In this walkthrough exercise, you start MarkLogic in your CentOS virtual machine.

 

  1. In your virtual machine, in a command prompt window, as the root user, enter /etc/init.d/MarkLogic start.
  2. See that MarkLogic has been started:

  3. Enter exit to log out as the root user.
  4. Close the command prompt window.
