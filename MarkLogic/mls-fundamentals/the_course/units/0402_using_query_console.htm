Using Query Console

Query Console is a web-based MarkLogic interface, available at localhost:8000, that allows you to execute JavaScript or XQuery expressions in MarkLogic.

To query a database, that database needs to be selected in the Content Source drop-down list.

The Content Source drop-down list can also be used to view the documents in a database by selecting the database and then clicking the Explore button.

For retrieving a subset of the records in a database, Query Console supports expressions written in JavaScript (general purpose), SPARQL (for querying RDF triples), SQL (for using MarkLogic with Tableau or Cognos), and XQuery (general purpose).

Expressions can also be written that do not interact with the selected database, such as running a script to create a new database.
