Downloading the Installer

MarkLogic can be downloaded from developer.marklogic.com/products.

The Downloads page contains the installation files for the supported operating systems.

Licensing

There are three types of MarkLogic licenses: Developer, Essential Enterprise, and Global Enterprise. Details for each license type can be found on the pricing page at www.marklogic.com/pricing.

The Developer License is for developers, and comes with all of the MarkLogic features. It can be clustered across multiple computers and has a 1TB size limitation. It is only for non-production use and has a six month time limit. New licenses may be requested on expiration.
