Starting the Service

To start MarkLogic on CentOS, the following command needs to be run by the root user:

/etc/init.d/MarkLogic start

All Platforms

The steps for starting MarkLogic on all platforms are available in the Starting MarkLogic Server section of the Installation Guide for All Platforms (docs.marklogic.com/guide/installation/procedures#id_92457).
