Exercise: See Semantics in Use

In this walkthrough exercise, you see semantics in a MarkLogic application.

 

  1. In a browser, navigate to mlu.marklogic.com/top-songs.
  2. On the left, for artist, select the beatles.
  3. Notice the list of suggestions, based on members in the band:

  4. In the search field, enter John Lennon.
  5. Select the [more] link for Whatever Gets You Thru the Night.
  6. On the song detail page, notice the artist name of John Lennon, in green text:


  7. Hover over John Lennon, to see semantic data for John Lennon display:


  8. Read through the semantic data for John Lennon (e.g. birth date, birth place, given name, surname, description, comments, and death date).
  9. In a browser, navigate to mlu.marklogic.com/ondemand.
  10. In the search field, enter semantics:

  11. Click the [More] link for Getting to Know MarkLogic Semantics.
  12. If desired, click the heading Getting to Know MarkLogic Semantics, to start watching the 40-minute on-demand video.
