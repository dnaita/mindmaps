Exercise: See How MarkLogic Is Being Used

In this walkthrough exercise, you access customer examples of how MarkLogic is being used.


  1. In a browser, navigate to www.marklogic.com/customers.
  2. Notice the alphabetized list of customers:


  3. Scroll down through the list, possibly seeing organizations that are familiar to you.
  4. Know that some MarkLogic customers are unlisted. (For example, government intelligence organizations and companies that choose to be unlisted for competitive reasons.)
  5. Select the Testimonials tab:


  6. Notice various testimonials (e.g. HealthCare.gov, Condé Nast, Wiley, Federal Aviation Administration, etc.).
  7. Select a testimonial that looks interesting to you (e.g. Condé Nast).
  8. Skim through the testimonial, noting why the organization chose MarkLogic.
  9. Click the browser's Back button, to return to the list of testimonials.
  10. Notice the testimonial for Yell.
  11. In a browser window, navigate to www.yell.com.
  12. Realize that this application uses MarkLogic for storing all of its data.
  13. In the search field, enter thai.
  14. Notice the autocomplete feature, retrieving values from the database:


  15. Select thai restaurants.
  16. Enter London for the city and click the Search button.
  17. See the results display, including the map:

  18. Realize that MarkLogic's underlying geospatial functionality is being used to provide the location of the restaurants.
  19. In a browser, navigate to developer.marklogic.com/blog/marklogic-bbc-roundup.
  20. Notice that this blog post contains a video demonstration of the 2012 BBC Olympics site, which uses MarkLogic to store and serve content:


  21. If desired, watch the first couple minutes of the video.
  22. Return to the MarkLogic customer testimonials page.
  23. Select the County of Fairfax logo:

  24. Select the Land Use Repository tab:


  25. Skim through the page, to understand the challenge and solution.
  26. In a browser, navigate to ldip.fairfaxcounty.gov.
  27. In the search fields, enter "swimming pools" for All text, 01/01/2014 for After Date, and 12/31/2014 for Before Date, to access all of the 2014 properties that reference a swimming pool:


  28. Click the Submit button to see the results:


  29. Notice that, because of MarkLogic's stemming feature for search, records with "swim pool" were also found.
  30. Continue perusing the testimonials as desired.
