Using Semantics

MarkLogic can be used to store RDF triples and process SPARQL queries.

RDF Triples

Semantic data is stored in RDF (Resource Description Framework) triples, which are composed of a subject, predicate, and object. RDF triples relate one item to another.

Triples can be loaded and stored in MarkLogic, and then accessed with SPARQL, for use in an application.

SPARQL

SPARQL (SPARQL Protocol and RDF Query Language) is the query language for retrieving and manipulating data stored in RDF triples.

xquery version "1.0-ml";
import module namespace sem = "http://marklogic.com/semantics" at "/MarkLogic/semantics.xqy";

let $artist := "The_Beatles"
let $bindings := map:map()
let $_binding := map:put($bindings, 'band', sem:iri(fn:concat("http://dbpedia.org/resource/", $artist)))
let $result :=
  sem:sparql("
    PREFIX rsrc: &lt;http://dbpedia.org/resource/&gt;
    PREFIX onto: &lt;http://dbpedia.org/ontology/&gt;

    ASK
    WHERE { $band onto:bandMember ?o }
    ", $bindings)

return ...
