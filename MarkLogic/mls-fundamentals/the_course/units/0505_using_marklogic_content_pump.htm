Using MarkLogic Content Pump to Bulk Load

MarkLogic Content Pump (MLCP) is a Java-based command line tool for bulk loading files into a MarkLogic database.

Setting up MLCP

As a Java-based command line tool, MLCP requires JRE 1.6 or later.

MLCP can be downloaded for free from developer.marklogic.com/products/mlcp. Once it has been downloaded, the files can be unpacked to any folder on the file system.

For MLCP to be able to communicate with a MarkLogic database, an XDBC server needs to be set up for the database. This can be done in the Admin Interface.

Using MLCP

To use MLCP, a command can be run on the command line, from the directory where MLCP is located.

For example, the following could be run in a command prompt window, to bulk upload documents on the file system to a database that has an XDBC server set up at port 8065.

mlcp.sh import -host localhost -port 8065 -username admin -password admin -mode local -input_file_path Desktop/mls-fundamentals/unit05/patents_data -output_uri_replace "home/cent/Desktop/mls-fundamentals/unit05/source, 'patents'"

Note: If using Windows, use mlcp.bat instead of mlcp.sh.
