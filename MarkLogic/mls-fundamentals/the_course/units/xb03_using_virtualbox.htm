Using VirtualBox

VirtualBox is the free virtualization software that was used to create the course virtual machine.

To run the course virtual machine, you need to have VirtualBox (free to download) installed on your computer.

Downloading VirtualBox

VirtualBox can be downloaded from www.virtualbox.org.

VirtualBox is available for the Windows, Macintosh, Linux, and Solaris operating systems.

Importing an .ova File

In VirtualBox, to import the course virtual machine image (i.e. the .ova file), the File - Import Appliance... command can be used.
