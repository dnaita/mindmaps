Exercise: Install MarkLogic

In this walkthrough exercise, you install MarkLogic on your virtual machine.

 

  1. In a browser, navigate to the Installation Guide for All Platforms at docs.marklogic.com/guide/installation.
  2. Click the Procedures link.
  3. Scroll down to the table that lists the operating systems and their corresponding steps.
  4. Scan the list, noticing that there are steps for installing on Window x64, Red Hat Linux 64, Sun Solaris x64, and Mac OS X.
  5. Note the Red Hat Linux x64 steps:

  6. In your virtual machine, double-click the MarkLogic-8.0-3-2.x86_64.rpm file:

  7. When prompted, click the Install button.
  8. Watch the installation progress.
  9. When prompted, enter centos as the password for root:


  10. Click the Authenticate button.
  11. Watch the installation complete.

Configure MarkLogic to run as cent:

  1. Select Applications - System Tools - Terminal, to open a command prompt window.
  2. Enter su - to become the root user.
  3. For password, enter centos.
  4. Enter vim /etc/sysconfig/MarkLogic.
  5. Use the down arrow key on your keyboard to navigate to export MARKLOGIC_USER=daemon (line 17).
  6. Use the right arrow key on your keyboard to navigate to the d in daemon.
  7. Press the i key on your keyboard, to be able to edit the document.
  8. Type cent.
  9. Press the delete key on your keyboard 6 times, to delete daemon.
  10. Press the Esc key on your keyboard.
  11. Type :wq and then press the Enter key, to exit.
  12. Leave the command prompt window open, for the next exercise.
