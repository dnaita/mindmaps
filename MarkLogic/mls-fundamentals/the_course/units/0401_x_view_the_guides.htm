Exercise: View the Guides

In this walkthrough exercise, you locate the guides on the various ways to communicate with MarkLogic.

 

  1. In a browser, navigate to docs.marklogic.com.
  2. See that you are on the landing page for the guides.
  3. In the Developer's Guides section, note the REST Application Developer's Guide, the Java Application Developer's Guide, the Node.js Application Developer's Guide, and the Application Developer's Guide (XQuery).
  4. In the Administrator's Guides section, note the Administrator's Guide and the Scripting Administrative Tasks Guide.
  5. Click the Scripting Administrative Tasks Guide heading.
  6. In the table of contents, click Using the Admin API.
  7. Skim through the beginning of the chapter, realizing that the Admin API provides a way to script administrative tasks, like those available in the Admin Interface GUI.
  8. Click the browser's Back button twice, to return to the list of guides.
  9. In the Other Reference Guides section, select the Query Console User Guide.
  10. Click Introduction to Query Console.
  11. Skim through the page to start learning about Query Console, a graphical interface that provides a way to write and execute scripts while interfacing with MarkLogic.
