Overview: MarkLogic Fundamentals

Length: 1 day

In this 1-day hands-on course, you learn the basics of MarkLogic, including what it is and what it's being used for, how to install it, how to create a database, how to insert data, how to define an application server and set up an application, how search works, how indexes can be used, as well as how additional technologies can be used with MarkLogic.

The recommended prerequisites for this course are any programming language, web development experience, and database experience.

 

Unit 1: Learning What MarkLogic Is
Unit 2: Installing MarkLogic
Unit 3: Creating a Database
Unit 4: Communicating with MarkLogic
Unit 5: Inserting Data
Unit 6: Setting Up an Application
Unit 7: Introducing Search
Unit 8: Adding Indexes
Unit 9: Using More Technologies

 

For live training online, you will want to have two computers (or one computer with two monitors) so that you can see what the instructor is doing on one monitor, while you are working on the second monitor.

For the self-paced version of this course, you will want to do the setup steps in the self-paced course videos to ensure that your computer will be able to run the course virtual machine (downloadable from MarkLogic University) locally.

This course was written on the CentOS platform, using a MarkLogic University virtual machine (for running on a Windows, Macintosh, Linux, or Solaris host computer).

This course could also potentially be done without the course virtual machine, directly on a Linux computer (for setup, see See the Steps That Were Done), or on a Macintosh or Windows (with adjustments as needed, such as file paths, and MarkLogic installation steps). For a list of operating systems supported by MarkLogic, please see the MarkLogic Installation Guide.
